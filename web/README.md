# Ling 文档站（web/）

本目录是灵轨（Ling）的 Next.js 文档站点源码，用于展示模板资产与使用说明。

## 本地开发

```bash
cd web
npm install
npm run dev
```

发布构建与代码检查：

```bash
npm run build
npm run lint
```
