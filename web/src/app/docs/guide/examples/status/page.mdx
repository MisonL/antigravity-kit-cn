# 项目状态

使用 `/status` 获取项目全局概览。

## 概览

`/status` 会提供项目的**完整仪表盘视图**。
你可以看到项目信息、技术栈、已完成的功能、智能体活动与预览状态。

  <Feature title="文件" description="统计数据" />
  <Feature title="预览" description="健康检查" />
</FeatureGrid>

## 过程

<StepList>
  <Step number={1} title="运行命令">
    直接运行即可查看项目仪表盘。

    ```bash
    /status
    ```
  </Step>
</StepList>
