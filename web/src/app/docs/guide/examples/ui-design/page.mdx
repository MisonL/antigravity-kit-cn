# 高级 UI 设计

使用 `/ui-ux-pro-max` 立刻生成高转化的落地页。

## 概览

这是一个专为**快速落地页开发**设计的标准化工作流。

<Callout type="info">
    **由 NextLevelBuilder 驱动：**该工作流与
    [nextlevelbuilder.io](https://ui-ux-pro-max-skill.nextlevelbuilder.io/#styles)
    的精选风格提示深度适配。你选择风格，AI 负责实现。
</Callout>

## 过程

<StepList>
  <Step number={1} title="选择风格">
    访问 [nextlevelbuilder.io](https://ui-ux-pro-max-skill.nextlevelbuilder.io/#styles) 并复制风格提示（如 "Modern SaaS - Dark Mode"）。
  </Step>
  <Step number={2} title="运行命令">
    粘贴风格提示与内容需求。

    ```bash
    /ui-ux-pro-max 创建一个友好的宠物美容服务落地页，使用粘土拟态卡片，
    包含服务套餐、宠物画廊、预约系统和宠物主人评价。
    使用温暖活泼的配色。
    ```

  </Step>
  <Step number={3} title="构建阶段">
    智能体自动搭建完整页面结构。

    <div className="grid grid-cols-1 md:grid-cols-3 gap-4 mt-4">
      <div className="bg-zinc-50 dark:bg-zinc-900 border border-zinc-200 dark:border-zinc-800 rounded-lg p-3">
        <h4 className="text-sm font-semibold mb-1">📐 结构</h4>
        <p className="text-xs text-zinc-500 dark:text-zinc-400 m-0">语义化 HTML5 与 SEO 层级</p>
      </div>
      <div className="bg-zinc-50 dark:bg-zinc-900 border border-zinc-200 dark:border-zinc-800 rounded-lg p-3">
        <h4 className="text-sm font-semibold mb-1">🧩 组件</h4>
        <p className="text-xs text-zinc-500 dark:text-zinc-400 m-0">可复用、可访问的 UI 模块</p>
      </div>
      <div className="bg-zinc-50 dark:bg-zinc-900 border border-zinc-200 dark:border-zinc-800 rounded-lg p-3">
        <h4 className="text-sm font-semibold mb-1">✨ 交互</h4>
        <p className="text-xs text-zinc-500 dark:text-zinc-400 m-0">微动效与响应式体验</p>
      </div>
    </div>

  </Step>
</StepList>
