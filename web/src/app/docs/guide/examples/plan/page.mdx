# 项目规划

在写代码前，使用 `/plan` 生成详细实施计划。

## 概览

`/plan` 会进入**仅规划模式**。
它使用 `project-planner` 生成详细计划文件，不写任何代码。

<Callout type="info">
    **关键规则：**规划阶段不写代码，只生成 `PLAN.md`。
</Callout>

## 过程

<StepList>
  <Step number={1} title="描述项目">
    告诉规划器你要构建什么。

    ```bash
    /plan 带购物车和结账的电商网站
    ```

  </Step>
  <Step number={2} title="苏格拉底式提问">
    规划器在生成计划前会先澄清问题。
  </Step>
  <Step number={3} title="生成计划">
    创建 `docs/PLAN-{slug}.md`，包含任务与智能体分配。
  </Step>
</StepList>
