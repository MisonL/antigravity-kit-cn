# 预览管理

使用 `/preview` 管理本地开发服务器。

## 概览

`/preview` 用于管理本地开发服务器。
你可以用简单命令启动、停止、重启或检查状态。

<FeatureGrid>
  <Feature title="start" description="启动服务" />
  <Feature title="stop" description="停止服务" />
  <Feature title="restart" description="重启" />
  <Feature title="check" description="健康检查" />
</FeatureGrid>

## 过程

<StepList>
  <Step number={1} title="启动服务">
    ```bash
    /preview start
    ```
  </Step>
  <Step number={2} title="检查状态">
    ```bash
    /preview
    ```
  </Step>
  <Step number={3} title="停止服务">
    ```bash
    /preview stop
    ```
  </Step>
</StepList>
