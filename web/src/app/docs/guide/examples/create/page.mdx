# 创建新应用

使用 `/create` 向导从零构建完整应用。

## 概览

`/create` 是你的**应用创建向导**。
它会引导你进行交互式对话，理解需求、规划架构，并编排专家智能体完成构建。

<Callout type="info">
    **从想法到可运行应用：**只需描述需求，向导会处理技术栈、目录结构、数据库
    schema 与实现细节。
</Callout>

## 过程

<StepList>
  <Step number={1} title="描述应用">
    告诉向导你要构建什么。如果信息不足，它会追问澄清。

    ```bash
    /create 带商品列表和购物车的电商网站
    ```

  </Step>
  <Step number={2} title="审核计划">
    `project-planner` 会生成详细计划，确认后进入执行。
  </Step>
  <Step number={3} title="智能体构建">
    专家智能体并行工作：database-architect、backend、frontend。
  </Step>
</StepList>
