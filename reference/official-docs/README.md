# 官方规则文档索引

本目录用于记录 Antigravity、Codex、Gemini CLI 的官方规则约定，作为本仓库的对照基线。

## 记录原则

1. 以官方文档原始地址为准，不编造机制。
2. 本目录可做中文整理，但不应改变官方规则语义。
3. 每次核对都写入证据索引（URL、时间、SHA/提交记录）。
