# SEO 基础 (SEO Fundamentals)

> 提升搜索引擎可见性的基本原则。

---

## 1. E-E-A-T 框架

| 原则                           | 信号                      |
| ------------------------------ | ------------------------- |
| **经验 (Experience)**          | 第一手经验、真实案例      |
| **专业性 (Expertise)**         | 资质证明、知识深度        |
| **权威性 (Authoritativeness)** | 反向链接、提及、行业认可  |
| **可靠性 (Trustworthiness)**   | HTTPS、透明度、信息准确性 |

---

## 2. 核心 Web 指标 (Core Web Vitals)

| 指标    | 目标    | 衡量维度   |
| ------- | ------- | ---------- |
| **LCP** | < 2.5s  | 加载性能   |
| **INP** | < 200ms | 交互性     |
| **CLS** | < 0.1   | 视觉稳定性 |

---

## 3. 技术 SEO 原则

### 网站结构

| 元素                      | 用途         |
| ------------------------- | ------------ |
| XML 站点地图              | 帮助抓取     |
| robots.txt                | 控制访问     |
| 规范标签 (Canonical tags) | 防止内容重复 |
| HTTPS                     | 安全信号     |

### 性能

| 因素       | 影响          |
| ---------- | ------------- |
| 页面速度   | 核心 Web 指标 |
| 移动端友好 | 排名因素      |
| 清晰的 URL | 可抓取性      |

---

## 4. 内容 SEO 原则

### 页面元素

| 元素       | 最佳实践                 |
| ---------- | ------------------------ |
| Title 标签 | 50-60 字符，关键词靠前   |
| Meta 描述  | 150-160 字符，具有吸引力 |
| H1         | 每页一个，包含主关键词   |
| H2-H6      | 逻辑层级清晰             |
| Alt 文本   | 描述性强，不堆砌关键词   |

### 内容质量

| 因素   | 重要性       |
| ------ | ------------ |
| 深度   | 全面覆盖主题 |
| 新鲜度 | 定期更新     |
| 独特性 | 具有原创价值 |
| 可读性 | 写作清晰易懂 |

## 5. 结构化数据 (Schema Markup) 类型

| 类型           | 用途           |
| -------------- | -------------- |
| Article        | 博客文章、新闻 |
| Organization   | 公司信息       |
| Person         | 作者个人资料   |
| FAQPage        | 问答内容       |
| Product        | 电子商务       |
| Review         | 评分           |
| BreadcrumbList | 导航面包屑     |

---

## 6. AI 内容指南

### Google 的关注点

| ✅ 要 (Do)         | ❌ 不要 (Don't)  |
| ------------------ | ---------------- |
| AI 初稿 + 人工编辑 | 发布原始 AI 内容 |
| 添加原创见解       | 无价值的复制     |
| 专家审核           | 跳过事实核查     |
| 遵循 E-E-A-T       | 堆砌关键词       |

---

## 7. 排名因素（按优先级排序）

| 优先级 | 因素                      |
| ------ | ------------------------- |
| 1      | 高质量、相关的内容        |
| 2      | 来自权威网站的反向链接    |
| 3      | 页面体验（核心 Web 指标） |
| 4      | 移动端优化                |
| 5      | 技术 SEO 基础             |

---

## 8. 衡量指标

| 指标          | 工具                   |
| ------------- | ---------------------- |
| 排名          | Search Console, Ahrefs |
| 流量          | Analytics              |
| 核心 Web 指标 | PageSpeed Insights     |
| 收录情况      | Search Console         |
| 反向链接      | Ahrefs, Semrush        |

---

> **记住：** SEO (搜索引擎优化) 是一场持久战。高质量内容 + 技术卓越 + 耐心 = 最终成果。
