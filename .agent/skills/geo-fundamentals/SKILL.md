# GEO Fundamentals - GEO 基础

> 针对 AI 驱动的搜索引擎的优化。

---

## 1. 什么是 GEO (Generative Engine Optimization)?

**GEO** = 生成引擎优化

| 目标               | 平台                                |
| ------------------ | ----------------------------------- |
| 在 AI 回复中被引用 | ChatGPT, Claude, Perplexity, Gemini |

### SEO vs GEO

| 方面 | SEO                | GEO        |
| ---- | ------------------ | ---------- |
| 目标 | 排名第一           | AI 引用    |
| 平台 | Google             | AI 引擎    |
| 指标 | 排名、点击率 (CTR) | 引用率     |
| 焦点 | 关键词             | 实体、数据 |

---

## 2. AI 引擎格局 (AI Engine Landscape)

| 引擎           | 引用风格        | 机会         |
| -------------- | --------------- | ------------ |
| **Perplexity** | 编号引用 [1][2] | 最高的引用率 |
| **ChatGPT**    | 行内/脚注       | 自定义 GPTs  |
| **Claude**     | 上下文式        | 长篇内容     |
| **Gemini**     | 来源部分        | SEO 交叉     |

---

## 3. RAG 检索因素 (Retrieval Factors)

AI 引擎如何选择要引用的内容：

| 因素       | 权重 |
| ---------- | ---- |
| 语义相关性 | ~40% |
| 关键词匹配 | ~20% |
| 权威信号   | ~15% |
| 新鲜度     | ~10% |
| 来源多样性 | ~15% |

---

## 4. 容易被引用的内容

| 元素             | 为什么有效           |
| ---------------- | -------------------- |
| **原始统计数据** | 独特的、可引用的数据 |
| **专家语录**     | 权威性转移           |
| **清晰的定义**   | 易于提取             |
| **分步指南**     | 可操作的价值         |
| **对比表**       | 结构化信息           |
| **FAQ 部分**     | 直接答案             |

---

## 5. GEO 内容检查清单 (Content Checklist)

### 内容元素

- [ ] 基于问题的标题
- [ ] 顶部的总结/TL;DR
- [ ] 带有来源的原始数据
- [ ] 专家语录 (姓名，头衔)
- [ ] FAQ 部分 (3-5 个问答)
- [ ] 清晰的定义
- [ ] "最后更新" 时间戳
- [ ] 带有凭证的作者信息

### 技术元素

- [ ] 带有日期的文章 Schema
- [ ] 作者的 Person Schema
- [ ] FAQPage Schema
- [ ] 快速加载 (< 2.5s)
- [ ] 干净的 HTML 结构

## 7. AI 爬虫访问 (AI Crawler Access)

### 关键 AI 用户代理 (User-Agents)

| 爬虫          | 引擎           |
| ------------- | -------------- |
| GPTBot        | ChatGPT/OpenAI |
| Claude-Web    | Claude         |

| 策略        | 何时使用           |
| ----------- | ------------------ |
| 允许所有    | 想要 AI 引用       |

## 8. 测量 (Measurement)

| 指标               | 如何追踪     |
| ------------------ | ------------ |
| AI 引用            | 手动监控     |
| "根据 [品牌]" 提及 | 在 AI 中搜索 |
| 竞争对手引用       | 比较份额     |
| AI 带来的流量      | UTM 参数     |

---

## 9. 反模式 (Anti-Patterns)

| ❌ 不要      | ✅ 要      |
| ------------ | ---------- |
| 发布不带日期 | 添加时间戳 |
| 模糊的归属   | 具名来源   |
| 跳过作者信息 | 显示凭证   |
| 内容单薄     | 全面覆盖   |

---

> **记住：** AI 引用清晰、权威且易于提取的内容。做最好的答案。
