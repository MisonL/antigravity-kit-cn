# GEO 基础

> 针对 AI 驱动的搜索引擎的优化。

---

## 1. 什么是 GEO？

**GEO** = Generative Engine Optimization (生成引擎优化)

| 目标               | 平台                                |
| ------------------ | ----------------------------------- |
| 在 AI 回复中被引用 | ChatGPT, Claude, Perplexity, Gemini |

### SEO vs GEO

| 方面 | SEO          | GEO                    |
| ---- | ------------ | ---------------------- |
| 目标 | 排名第一     | AI 引用                |
| 平台 | Google       | AI 引擎 (AI engines)   |
| 指标 | 排名、点击率 | 引用率 (Citation rate) |
| 焦点 | 关键词       | 实体、数据             |

---

## 2. AI 引擎格局

| 引擎           | 引用风格    | 机会       |
| -------------- | ----------- | ---------- |
| **Perplexity** | 编号 [1][2] | 最高引用率 |
| **ChatGPT**    | 行内/脚注   | 自定义 GPT |
| **Claude**     | 上下文      | 长篇内容   |
| **Gemini**     | 来源部分    | SEO 交叉   |

---

## 3. RAG 检索因素

AI 引擎如何选择内容进行引用：

| 因素                            | 权重 |
| ------------------------------- | ---- |
| 语义相关性 (Semantic relevance) | ~40% |
| 关键词匹配                      | ~20% |
| 权威信号                        | ~15% |
| 新鲜度                          | ~10% |
| 来源多样性                      | ~15% |

---

## 4. 被引用的内容

| 元素             | 为什么有效         |
| ---------------- | ------------------ |
| **原始统计数据** | 独特且可引用的数据 |
| **专家引用**     | 权威传递           |
| **清晰定义**     | 易于提取           |
| **分步指南**     | 可操作的价值       |
| **对比表**       | 结构化信息         |
| **常见问题部分** | 直接回答           |

---

## 5. GEO 内容检查清单

### 内容元素

- [ ] 基于问题的标题
- [ ] 顶部摘要/太长不看 (TL;DR)
- [ ] 带来源的原始数据
- [ ] 专家引用（姓名、头衔）
- [ ] 常见问题部分（3-5 个问答）
- [ ] 清晰定义
- [ ] "最后更新"时间戳
- [ ] 带资质的作者

### 技术元素

- [ ] 带日期的文章 Schema
- [ ] 作者的人物 Schema
- [ ] FAQPage schema
- [ ] 快速加载 (< 2.5s)
- [ ] 清晰的 HTML 结构

## 7. AI 爬虫访问

### 关键 AI 用户代理 (User-Agents)

| 爬虫          | 引擎           |
| ------------- | -------------- |
| GPTBot        | ChatGPT/OpenAI |
| Claude-Web    | Claude         |
| PerplexityBot | Perplexity     |
| Googlebot     | Gemini（共享） |

### 访问决策

| 策略        | 何时使用           |
| ----------- | ------------------ |
| 允许所有    | 想要 AI 引用       |
| 阻止 GPTBot | 不想要 OpenAI 训练 |

## 8. 测量

| 指标               | 如何追踪     |
| ------------------ | ------------ |
| AI 引用            | 手动监控     |

---

## 9. 反模式

| ❌ 不要      | ✅ 要      |
| ------------ | ---------- |
| 发布不带日期 | 添加时间戳 |
| 模糊归属     | 命名来源   |
| 跳过作者信息 | 显示资质   |
| 内容单薄     | 全面覆盖   |

---

> **记住：** AI 引用那些清晰、权威且易于提取的内容。成为最好的答案。
