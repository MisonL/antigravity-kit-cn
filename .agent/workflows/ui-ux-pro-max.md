# 极致UI (UI/UX Pro Max Workflow)

**触发命令**: `/ui-ux-pro-max [description]`

## 目的

专为追求**极致美学**和**高级交互**的场景设计。拒绝平庸的 Bootstrap 风格，打造 Dribbble/Awwwards 级别的界面。

## 核心能力

1. **风格引擎**:
    - 内置 50+ 种设计风格 (Glassmorphism, Neo-Brutalism, Minimalist, Cyberpunk...)。
    - 21+ 种精心调配的色板。
    - 50+ 种字体组合。

2. **步骤流程**:
    - **风格选择**: 让用户选择或 AI 推荐风格。
    - **组件定制**: 定制 Shadcn/UI 组件以匹配风格（修改圆角、阴影、动画）。
    - **微交互**: 添加 Framer Motion 动画 (Hover, Tap, In-View)。
    - **布局实现**: 使用 CSS Grid/Flexbox 实现响应式杂质布局。

3. **设计原则**:
    - **留白**: 大胆使用留白。
    - **排版**: 建立清晰的文字层级。
    - **对比**: 确保视觉焦点突出。

## 示例

> User: /ui-ux-pro-max 这是一个面向设计师的个人作品集
> AI: 明白。推荐使用 **Minimalist (极简)** 风格，配合 **Inter** 字体和 **黑白+电光蓝** 配色。正在生成具有视差滚动效果的首页...
